# 🧁 Cupcake de Café com Chantilly

Projeto de página de receita desenvolvido com base em um protótipo do Figma durante estudos na Rocketseat.

## 🚀 Tecnologias

* HTML5
* CSS3 (variáveis, pseudo-classes e layout básico)
* Figma

## 🎨 Sobre o projeto

A aplicação apresenta uma receita completa de cupcake, com foco em organização visual e fidelidade ao design proposto.

### Funcionalidades:

* Estrutura semântica em HTML
* Estilização com CSS
* Layout centralizado
* Lista de ingredientes estilizada
* Passo a passo de preparo

## 🛠️ Melhorias implementadas

* Organização do código CSS
* Ajustes visuais no layout

## ▶️ Como executar

```bash
git clone https://github.com/rodrigomscmontenegro/projeto-receitas
cd projeto-receitas
```

Abra o arquivo `index.html` no navegador.

## 🎯 Objetivo

Praticar conceitos básicos de HTML e CSS, além da interpretação de layouts no Figma.
