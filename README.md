# 🧁 Cupcake de Café com Chantilly

Este é um projeto de uma página de receitas, desenvolvida durante o curso da Rocketseat.

## 🚀 Tecnologias
- HTML5
- CSS3 (Variáveis, seletores avançados e Google Fonts)
- Figma (Prototipação)

## 🎨 Layout
O layout foi baseado no design do Figma. A página apresenta:
- Cartão de conteúdo centralizado.
- Lista de ingredientes estilizada.
- Instruções detalhadas de preparo.

## 🛠️ Como visualizar
1. Clone o repositório.
2. Abra o arquivo `index.html` no seu navegador.
